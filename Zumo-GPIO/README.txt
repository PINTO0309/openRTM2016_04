このフォルダには，RTミドルウェアコンテスト2016に参加している「ZumoとRaspberry Piを用いた教育用ロボット環境」のZumo ロボット内制御プログラム (GPIOシリアル通信版）が保存されています．作品は以下のファイルから構成されています．

(1) Zumo ロボット用 Arduino統合環境設定ファイル 
(2) ロボット制御用C++ソース，ヘッダファイル群

（注）
このロボット内制御プログラムは，Zumoロボット基板上のGPIOシリアル通信を利用の場合（Raspberry Pi A+を想定）用です．
Zumoロボット内制御プログラムには，Zumoロボット基板上のGPIOシリアル通信を利用の場合（Raspberry Pi A+を想定），Micro USBポート経由でのシリアル通信（通常はこちらを使用）の2種類があります．ご使用の環境に応じて，プログラムを使い分ける必要があります．

フォルダ，ファイルの一覧は以下のとおりです．

- README.txt		README
- Zumo-rasp.ino		Pololu Zumo 32U4用プログラム Arduino統合環境 設定ファイル
			（Zumoロボット基板上のGPIOシリアル通信を利用）
- *.cpp, *.h		モータ，ボタン，シリアル通信用プログラム

これらの，フォルダ，ファイルを利用して，Zumoロボットのアプリケーションソフトウェアの開発が行えます．Zumoロボットそのもの機能の追加には，このプログラム群を修正，改良して，開発を行います．
